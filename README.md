#

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## 追加する機能
* 主人公と敵の技・MPの実装
* BGM, SE
* キャラの描画・エフェクト
* レベルアップ・セーブ
* ステージ選択

## ゲームの概要
* 勇者が魔王を倒すノンフィールドRPG
* 基本機能の参考にしたAI：Gemini

## ゲームの遊び方
* まず1/2/3キーを押してステージを選択する
* SPACEキーで次のターンへ
* 敵を倒すとステータスが上がる

## ゲームの実装
### 共通基本機能
* AIが作ったSPACEキーによりターンが進むだけのRPG

### 分担追加機能
* ステージ選択・音・タイトル画面・クレジット画面・隠しコマンド（担当：C0A24093）：ステージを選択して、5種類の敵と戦えるようにする。BGMと効果音を設定する。
* ゲームUIの変更
* 技エフェクトの描画
* HPバーの描画
* 勇者の行動選択画面の作成
