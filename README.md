## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## 追加する機能
* 主人公と敵の技・MPの実装
* BGM, SE
* キャラの描画・エフェクト
* レベルアップ・セーブ
* ステージ選択

## ゲームの概要
* 勇者を操作し、次々と現れる5体の敵を倒していくコマンド選択型（スペースキー進行）のテキストバトルRPGです。

## ゲームの遊び方
* まず1/2/3/4/5キーを押してステージを選択する
* A[Attack]で攻撃,H[Heal]で回復,D[defend]で防御,R[Run]で逃走
* プレイヤーの攻撃と敵の攻撃が交互に繰り出されます。
* 画面下のメッセージウィンドウにバトルのログが表示されます。
* 敵のHPを0にするとステージクリアとなり、rキーで次のステージ選択画面へ移行します。
* 勇者のHPが0になるとゲームオーバーです。


## ゲームの実装
### 共通基本機能
* スペースキーを押すと、プレイヤーの攻撃と敵の攻撃が交互に繰り出す。

### 分担追加機能
* ステージに応じた背景画像の切り替え、テキストボックスの改善　(C0A24245)
* ステージ選択・音・タイトル画面（担当：C0A24093）：ステージを選択して、5種類の敵と戦えるようにする。BGMと効果音を設定する。
* 防御・回復・逃走コマンドを追加(担当:C0A24237)
* 技エフェクトの描画、キャラクターの描画（担当：C0A24281）
* HPバーの描画・EXP機能（担当：C0A24125）：最大HPと現在のHPからHPバーを作成する関数、必要EXPと現在のEXPからEXPバーを作成・Levelアップを判定し、レベルアップ時にステータスを強化する関数
